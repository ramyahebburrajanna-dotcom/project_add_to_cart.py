# project_add_to_cart.py
Project 03 - Logout Automation using Selenium & Python
📖 Project Overview

This project automates the logout functionality of the SauceDemo application using Selenium WebDriver with Python.

The script performs the following tasks:

Opens the SauceDemo website
Logs in using valid credentials
Opens the navigation menu
Clicks the Logout option
Verifies that the user is redirected to the login page
Closes the browser
